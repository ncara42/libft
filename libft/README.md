DESCRIPCIÓN: Consiste en crear una librería propia, reimplementando funciones de la librería estándar (strlen, strcpy, memset, etc.) y creando otras adicionales útiles para los próximos proyectos. Todas estas funciones serán compiladas, tras haber sido convertidas de '.c' a '.o', en la librería final (en este caso, un archivo: libft.a).

INSTRUCCIONES: Existen varios comandos ejecutables para realizar distintas funciones.
	       
	       Comandos existentes (4): 'make', 'make clean', 'make fclean', 'make re'.
	
	       MAKE:  		Compila todas las funciones en un archivo '.a'.
               MAKE CLEAN:	Elimina los archivos '.o'.
	       MAKE FCLEAN:	Elimina los archivos '.o' y la librería.
  	       MAKE RE:		Ejecuta automaticamente 'clean' y 'fclean' y construye de nuevo la librería.

RECURSOS: Stackoverflow, documentación de C e IA. Se usaron para manejo de errores y sugerencias de optimización y mejora de legibilidad del código.
